# 人生脚本 v0.3 - 完整的剧本时间管理工具

## 🎬 核心理念

基于你的"剧本时间管理法"设计：
- **固定80%的日常动作** - 建立秩序感和确定性
- **从日到周到月** - 多维度的时间视图
- **执行情况标记** - 清晰的反馈机制
- **AI 智慧分析** - 基于数据的改进建议

---

## 📊 四大类别

你的每一个时间段都被归类为：

| 类别 | 颜色 | 说明 | 示例 |
|------|------|------|------|
| 🔵 健康习惯 | 蓝色 | 运动、冥想、睡眠 | 跑步、瑜伽、早睡 |
| 🟡 家庭时间 | 黄色 | 陪伴家人、家务 | 陪孩子、和父母 |
| 💚 工作时间 | 绿色 | 工作、学习、创作 | 写作、项目、工作 |
| 💜 自我成长 | 紫色 | 阅读、思考、提升 | 阅读、冥想日记 |

## 📅 功能使用

### 1️⃣ 剧本安排（今日视图）

**安排你的一天：**

1. 点击某个时间段（如 7:00）
2. 选择类别（如 🔵 健康习惯）
3. 输入活动标签（如 "冥想"、"跑步"）
4. 点击"确认"

**示例安排：**
```
7:00  → 🔵 冥想
8:00  → 🔵 健走
9:00  → 💚 工作
12:00 → 🟡 午餐陪孩子
14:00 → 💚 写作
18:00 → 🟡 家务
20:00 → 💜 阅读
```

### 2️⃣ 执行标记（完成情况）

点击时间段右侧的圆形按钮循环标记：

- ⭕ **计划中** - 还未执行
- ✅ **已完成** - 按计划完成
- ❌ **未完成** - 计划内但未执行

**示例：**
```
如果你早上7点安排了冥想，
晨起后冥想了 → 点击变成 ✅
或者因为起晚没冥想 → 点击变成 ❌
```

### 3️⃣ 周视图（整周一览）

点击底部 **"📊 周视图"** 查看：

- 纵轴：7:00 - 23:00 的每个小时
- 横轴：本周7天
- 单元格颜色：对应类别的颜色
- 单元格内容：✅ ❌ 的完成标记

**一眼看出：**
- 本周时间分配是否均衡？
- 哪天的执行效果最好？
- 哪个类别完成率最低？

### 4️⃣ 周复盘（反思与优化）

点击底部 **"📝 复盘"**：

**填写三个内容：**

1. **✅ 本周完成得很好的事**
   ```
   例：坚持了晨练，连续7天都没有缺席
       每个工作日都有1小时的专注写作时间
       每周陪孩子的时间增加了
   ```

2. **❌ 本周没做好的地方**
   ```
   例：周三加班到晚上，没有冥想
       阅读计划只完成了50%
       家务没有按计划完成
   ```

3. **💡 下周改进方案**
   ```
   例：周三调整工作时间，确保冥想时间不被占用
       增加阅读时间的灵活性（可以利用碎片时间）
       周末集中完成家务
   ```

### 5️⃣ AI 智慧分析

填写完复盘内容后，点击 **"✨ 获取本周分析"**

DeepSeek AI 会基于：
- 📊 你各类别的**完成率统计**
- 📝 你的**复盘反思**
- 🎬 **人生脚本**的理念

生成**专业的改进建议**，包括：
- 本周时间分配的评价
- 执行率最低类别的原因分析
- 下周的具体优化方案

## 🎯 最佳实践

### 每周的工作流

**周日晚上（制定剧本）**
```
1. 为下周的每个时间段安排活动
2. 确保四大类别的时间均衡
3. 保持80%的固定动作不变
```

**工作日（按剧本执行）**
```
1. 每完成一个时间段，就标记为 ✅
2. 如果没完成，标记为 ❌ 但继续执行下个时段
3. 不要纠结于单一失败，保持从容
```

**周末（复盘与优化）**
```
1. 填写周复盘内容
2. 获取 AI 分析
3. 基于分析调整下周计划
```

### 应对突发情况

**如果临时有事（如加班）：**
```
✅ 继续执行剧本的下一个环节
✅ 可以调整地点和方式，但保持动作一致
❌ 不要完全放弃，影响其他类别

例：本该在办公室写作，但临时出差
    → 在出差地点继续写作就可以
    → 标记为 ✅ 继续演好自己
```

## 🆘 常见问题

**Q：忘记同步，换设备丢失数据？**  
A：浏览器本地存储会保留，只要访问同网址就能恢复

**Q：如何重置本周数据？**  
A：浏览器清除缓存 或 删除 `localStorage` 中的 `life-script-week`

**Q：可以查看历史周数据？**  
A：目前支持当周数据，后续版本会加入多周对比功能

**Q：AI 分析不准确？**  
A：越详细的复盘内容，AI 给出的建议越准确
